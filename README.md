# React Utility Suite

A futuristic, interactive **React-based utility application** featuring an **AI-powered Translator** and a **Random String Generator**. Built with **React**, **Tailwind CSS**, **Framer Motion**, and **RapidAPI**, this project showcases modern front-end design with animations, dark/light mode, and user-friendly features.
